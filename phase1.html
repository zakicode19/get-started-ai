Phase 1: Foundations of AI and Programming

Goal: Establish a strong base in programming, mathematics, and fundamental AI concepts.

Learning Objectives

  • Master Python programming fundamentals for data science
  • Understand the mathematical foundations (linear algebra, probability, statistics) essential for AI
  • Grasp core machine learning concepts and algorithms
  • Become proficient in using Git for version control

Key Topics

Python Programming

Syntax, data structures, algorithms

Linear Algebra for AI

Vectors, matrices, eigenvalues

Probability and Statistics

Distributions, hypothesis testing, Bayesian thinking

Machine Learning Basics

Supervised and unsupervised learning

Version Control

Git fundamentals and best practices

Recommended Resources

Python

  • Python for Data Science (Coursera/edX)
  • "Python Crash Course" by Eric Matthes

Linear Algebra

  • "Essence of Linear Algebra" (3Blue1Brown YouTube series)
  • "Linear Algebra and Its Applications" by Gilbert Strang

Probability & Statistics

  • "Think Stats" by Allen B. Downey
  • Khan Academy: Probability and Statistics

Machine Learning

  • "Machine Learning" by Andrew Ng (Coursera)
  • "Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow" by Aurélien Géron

Git

  • Git Handbook (GitHub Guides)
  • "Pro Git" by Scott Chacon and Ben Straub

Practical Exercises

  • Solve programming challenges on platforms like LeetCode or HackerRank focusing on data structures and algorithms
  • Implement basic machine learning algorithms (e.g., linear regression, k-NN) from scratch in Python
  • Use Git to manage personal projects, including branching, merging, and pull requests

Phase 2: Core Natural Language Processing

Goal: Understand the principles and techniques of NLP, from text preprocessing to traditional NLP models.

Learning Objectives

  • Understand the unique challenges of processing human language
  • Master various text preprocessing techniques
  • Gain proficiency in traditional NLP models and their applications
  • Learn to represent words as numerical vectors using different embedding techniques
  • Become familiar with key NLP libraries like NLTK and SpaCy

Key Topics

Text Preprocessing

Tokenization, stemming, lemmatization, stop words, cleaning

Feature Engineering for Text

Bag-of-Words, TF-IDF, n-grams

Traditional NLP Models

Naive Bayes, Logistic Regression, SVMs for text classification

Word Embeddings

Word2Vec, GloVe, FastText

NLP Libraries

NLTK, SpaCy

Practical Applications

  • Text classification (sentiment analysis, spam detection)
  • Named Entity Recognition (NER)
  • Part-of-Speech (POS) tagging

Recommended Resources

Text Preprocessing & Traditional NLP

  • "Speech and Language Processing" by Daniel Jurafsky and James H. Martin
  • NLTK Book (online)
  • SpaCy documentation and tutorials

Word Embeddings

  • Word2Vec research paper
  • GloVe: Global Vectors for Word Representation
  • FastText official website

Practical Exercises

  • Perform text cleaning and preprocessing on a raw text dataset (e.g., tweets, news articles)
  • Build a spam classifier using Naive Bayes or Logistic Regression
  • Experiment with different word embedding models and visualize their relationships
  • Use NLTK and SpaCy for tasks like tokenization, POS tagging, and named entity recognition
